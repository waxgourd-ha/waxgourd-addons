# Node-RED 汉化版

基于物联网的可视化流程编程工具（汉化增强版）。

## 关于

[Node-RED][nodered] 是一款面向物联网的流程编程工具，提供基于浏览器的可视化编辑器，
方便地将各种节点连接成流程并一键部署。

本插件为汉化增强版，在官方 Node-RED HA 插件基础上，对编辑器 UI、内置节点及常用第三方插件进行了全面中文汉化。

## 汉化内容

- Node-RED 核心编辑器及内置节点
- node-red-dashboard（仪表板）
- node-red-contrib-modbus
- node-red-contrib-home-assistant-websocket
- node-red-contrib-bigtimer / moment / sunevents
- node-red-node-email / feedparser / suncalc / twitter
