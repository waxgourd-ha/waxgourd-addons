# 网心云容器魔方

**此版本为HA下专用，建议使用选择高速USB 3.0作为存储。打开选项，选择相应的分区**

默认使用内部存储，但小于30G的空间时建议外挂高速USB存储设备在USB3.0接口上。

如果使用高速U盘的话，建议读速超过200M，写入速度在100M以上。

小技巧：建议使用Disk Genius等软件，把U盘格式化成ext4会更稳定。
