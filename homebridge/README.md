# 冬瓜甄选addons：Homebridge

## 关于

Homebridge 是一款轻量级的Node.js服务器，你可以在家庭网络上运行，模拟 iOS HomeKit API。它支持插件，这些插件是由社区贡献的模块，提供从 HomeKit 到“智能家居”设备制造商提供的各种第三方 API 的基本桥梁。

由于Siri支持通过HomeKit添加的设备，这意味着通过Homebridge你可以让Siri控制完全不支持HomeKit的设备。例如，仅使用部分可用插件，你可以说：

- Siri，打开后门。
- Siri，开车库门。
- Siri，打开咖啡机。
- Siri，把客厅的灯打开。
- Siri，早上好！
